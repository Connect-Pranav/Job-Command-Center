# 🚀 JobHQ — GitHub Pages Deployment Guide
**Access your Job Command Center from any device, anywhere**

---

## Step 1 — Create a GitHub Account (if needed)
Go to **github.com** → Sign Up → use your personal email.

---

## Step 2 — Create a New Repository

1. Click the **+** icon (top-right) → **New repository**
2. Repository name: `jobhq` (or `job-command-center`)
3. Set to **Public** ✅ (required for free GitHub Pages)
4. Check **"Add a README file"**
5. Click **Create repository**

---

## Step 3 — Upload Your File

1. In your new repo, click **Add file → Upload files**
2. Drag and drop `job_command_center_v3.html`
3. **Rename it to `index.html`** (important — this makes it your homepage)
4. Scroll down → Click **Commit changes**

---

## Step 4 — Enable GitHub Pages

1. Go to your repo **Settings** (top tab)
2. Left sidebar → scroll to **Pages**
3. Under "Source" → select **Deploy from a branch**
4. Branch: **main** | Folder: **/ (root)**
5. Click **Save**

---

## Step 5 — Get Your Live URL

After 1–2 minutes, your URL will be:
```
https://YOUR-GITHUB-USERNAME.github.io/jobhq/
```

Example: `https://connect-pranav.github.io/jobhq/`

**Bookmark this.** Works on mobile, tablet, desktop — no app install needed.

---

## Step 6 — Add to Your Phone Home Screen

### iPhone (Safari):
1. Open the URL in Safari
2. Tap the **Share** button (box with arrow)
3. Tap **"Add to Home Screen"**
4. Name it **JobHQ** → Add
5. It now appears as an app icon on your home screen ✅

### Android (Chrome):
1. Open the URL in Chrome
2. Tap the **⋮ menu** (top right)
3. Tap **"Add to Home Screen"** or **"Install app"**
4. Confirm → Done ✅

---

## How to Update the File Later

When you want to update your JobHQ (e.g. add new companies, update your resume, etc.):

1. Go to your GitHub repo
2. Click on `index.html`
3. Click the **pencil icon** (Edit)
4. Make changes OR click **"Upload files"** to replace entirely
5. Commit changes → GitHub Pages auto-updates within 1-2 min

## Troubleshooting

**"404 Not Found"** → Make sure the file is named exactly `index.html` (not `job_command_center_v3.html`)

**Page not updating** → GitHub Pages can take up to 5 min. Hard refresh: `Ctrl+Shift+R` (PC) or `Cmd+Shift+R` (Mac)

**API calls not working on mobile** → This is normal — Claude API calls require internet. Make sure you're not in airplane mode.
